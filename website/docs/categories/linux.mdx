# Linux Certifications

Linux certifications demonstrate proficiency with open-source operating systems — from basic command-line skills through enterprise system administration and kernel internals.

## What You'll Find Here

- **Foundation-level** — Linux Essentials (LPI), CompTIA Linux+, and LFCS.
- **Professional-level** — LPIC-2, RHCSA, RHCE, and LFCE.
- **Specialist tracks** — LPIC-3 (Security, Virtualisation, Mixed Environments), and cloud-native Linux skills.
- **Curated resources** — interactive terminals, lab exercises, cheat sheets, and recommended reading.
