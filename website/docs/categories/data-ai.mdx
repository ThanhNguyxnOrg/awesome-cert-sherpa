# Data & AI Certifications

Data and AI certifications cover the full spectrum from database administration and analytics to machine learning engineering and AI ethics. This category tracks credentials that prove you can work with data at scale.

## What You'll Find Here

- **Data engineering** — cloud data platform certs, ETL/ELT pipeline credentials, and data warehousing tracks.
- **Data science & ML** — TensorFlow Developer, AWS ML Specialty, Azure AI Engineer, and vendor-neutral options.
- **Database administration** — Oracle DBA, Azure Database Administrator, MongoDB certs, and more.
- **Analytics & BI** — Power BI, Tableau, Google Data Analytics, and related credentials.
- **Curated resources** — datasets for practice, notebook tutorials, and community study groups.
